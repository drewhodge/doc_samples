Creating a DoDAF modeling project

You can open, close, save, and delete DoDAF projects and the files that belong to a project, including models. When you create a DoDAF modeling project, you can create both a new model and a set of DoDAF views.

Before you can create a DoDAF modeling project, enable the DoDAF modeling capabilities.
To create a new DoDAF modeling project:
  1. Click File > New > Project.
  2. In the New Project wizard, on the Select a wizard page, expand DoDAF, click DoDAF Project, and click Next.
  3. On the DoDAF Modeling Project page, type a project name, specify a workspace directory, and click Next.
  4. On the Create DoDAF Model page, from the File types list, select UML Modeling.
  5. From the Templates list, select DoDAF Model.
  6. In the File name field, type a name for the model.
  7. Click Finish.
  8. If the Confirm Perspective Switch window opens, click Yes.
In the Modeling perspective, the Model Explorer view displays a new project, a DoDAF Model (.emx) file, a blank model, and a hierarchy of DoDAF views.
