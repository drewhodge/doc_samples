Viewing architectural discovery results

You can view the results of an architectural discovery analysis in the Analysis Results view. You can filter the results to display results that correspond to each rule in your analysis configuration, or only those results that do not conform to a particular rule. From the Analysis Results view, you can also open results to study the relationships between the components of your code.

To view architectural discovery results, you must run an architectural discovery analysis.
To view the results of an architectural discovery analysis:
In the Analysis Results view, expand the top-level node of the tree that corresponds to your analysis configuration and complete one of the following steps:
  • To view only the results that do not conform to a particular rule, right-click anywhere in the view; then click Show Results Only.
  • To list the resources associated with each rule, expand the rule categories and rule nodes.
  • To view a visualization of a result, right-click the result; then click View ResultExplorer.
  • To see a component of the visualization, such as a class, an interface, or a package in the Package Exploer view, right-click the component; and then click Show in Package Explorer.
The results are listed in chronological order and are grouped into the same categories that you specified in the analysis configuration.
