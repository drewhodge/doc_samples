Enabling capabilities for static analysis

The Javaâ„¢ and Debug perspectives include analysis capabilities by default, but you can add analysis capabilities to any other perspective if you want to run a static analysis on resources other than Java source code files.

To add analysis capabilities to a perspective:
  1. Click Window > Open Perspective; then click a perspective to customize.
  2. Click Window > Customize Perspective.
  3. In the Customize Perspective window, click the Commands tab.
  4. In the Available command groups pane, select the Analysis check box.
