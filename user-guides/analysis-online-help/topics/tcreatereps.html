Creating reports from static analysis results

After you run an analysis configuration and generate results, you can capture the results in HTML-format reports.

To generate a report:
  1. In the Analysis Results view, expand the tree to the analysis provider level. This action enables the Report push button.
  2. Click Report .
  3. In the Analysis Reporting window, click a report type and click OK . The report is generated and displayed in the Web Browser view.
  4. To save the report, click File > Save As , select a destination folder and a file name with an .html extension, and click OK . The HTML report is saved with the required images for viewing or for publishing to a Web server.
